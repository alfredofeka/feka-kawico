Como sei se iniciei sessão (estou online)?

No canto superior direito da página principal (junto ao menu), você verá um indicador de status. Se estiver autenticado, o círculo estará **verde** e aparecerá a mensagem "Olá, [Seu Nome]". Se estiver desconectado, o círculo estará **amarelo** e aparecerá a palavra "Login".

Qual é o meu Username e a minha Password?

O seu **Username** foi gerado automaticamente quando se inscreveu (baseado no seu nome, ex: `joao01`). A sua **Password** é a que definiu durante o registo.

Ambos foram guardados com sucesso no seu navegador aquando da sua inscrição geral.

Esqueci a minha Password, como posso trocá-la?

Como não usamos um sistema de email, a recuperação exige a sua identidade:

  1. Vá para a página Trocar Password.
  2. Deve fornecer o **Email** e o **País de Registo**.
  3. Deve responder corretamente à sua **Pergunta Secreta** (ex: Qual o nome do seu primeiro animal de estimação?).
  4. Se todas as informações estiverem corretas, poderá definir uma nova Password instantaneamente.
Como posso partilhar um curso com um amigo?

É fácil! Em cada cartão de curso na página principal, existem pequenos botões no fundo:

  • : Partilha a mensagem do curso no WhatsApp.
  • : Partilha o link do curso no Facebook.
  • : Abre o seu cliente de email para enviar o link do curso a um amigo.
Por que preciso do LocalStorage (Armazenamento Local)?

O Local Storage é uma forma de guardar dados no seu próprio navegador. Como este site não tem um servidor (Backend) para guardar as suas inscrições, usamos o Local Storage para guardar os seus dados de registo e login.

IMPORTANTE: Se limpar os dados do seu navegador (cache e cookies), os seus dados de login serão perdidos e terá de se registar novamente.
